武器防具の売却




■売却で獲得できるアイテム

武器によって、売却した際にアイテムを獲得できます。

・★5武器を売却した場合
　パワーピース×1を獲得できます。
　限界突破済みの場合、限界突破の段階×1、
　獲得個数が増加します。 (最大4個)
　※レアリティ上昇で生成した
　　★5武器の売却では獲得できません。

・専用FR武器を売却した場合
　ハイパワーピース×1を獲得できます。
　限界突破済みの場合、限界突破の段階×1、
　獲得個数が増加します。 (最大4個)

・専用BT武器を売却した場合
　バーストパワーメダル×5を獲得できます。
　専用BT武器は限界突破が存在しないため、
　獲得個数は増加しません。

・武器、防具を売却した場合
　精錬に使用できる素材を獲得できます。
