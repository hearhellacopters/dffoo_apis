Additional Premium Rewards

Certain missions are eligible for additional premium rewards.

These rewards become available with the purchase of a Premium Moogle Pass.


Premium rewards for missions started during a Premium Moogle Pass's active period may still be collected after the Premium Moogle Pass expires.

For missions completed before purchasing a Premium Moogle Pass, premuim rewards can also be collected if a Premium Moogle Pass is purchased before the mission period ends.

Refer to the game guide below for more details on Premium Moogle Passes.
